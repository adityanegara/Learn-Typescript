A) Penjelasan Typescript

    Microsoft untuk menangani kekurangan JavaScript.

    dynamically typed ialah kebalikannya. Type data dapat dirubah.

    B.2) Code completion
    B.3) Refactoring
    B.4) Shorthand notations
